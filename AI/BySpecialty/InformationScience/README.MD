## 🌟 はじめに - なぜ情報科学がAIの心臓なのか

**日常例で理解する：**
レストランのシェフを想像してください。料理（AI）を作るには：
- 🍅 **材料の選び方**（データ処理）
- 🔪 **調理技術**（アルゴリズム）
- 📖 **レシピ理論**（理論的基盤）

情報科学は、この「料理学」全体を体系化した学問です。AIという料理を美味しく作るための、科学と技術の両面を提供します。

```mermaid
graph LR
    A[🍳 情報科学] --> B[🧠 理論的基盤<br/>数学・論理]
    A --> C[💾 データ処理<br/>情報の加工]
    A --> D[⚙️ アルゴリズム開発<br/>賢い手順]
    
    B --> E[🤖 人工知能の実現]
    C --> E
    D --> E
    
    style A fill:#ff9999
    style E fill:#99ccff
```

**なぜ重要？**
- AIは「勝手に賢くなる魔法」ではない → 設計図が必要
- 情報科学がなければ、AIは暗闇で料理するようなもの
- ChatGPTも自動運転も、全て情報科学の原理で動いている

## 📜 時代背景と発見に至った経緯

### 🎭 情報科学誕生の物語

```mermaid
timeline
    title 情報科学の進化 - AIへの道
    1930年代 : チューリングの計算理論<br/>🧮「機械は思考できるか？」
    1940年代 : シャノンの情報理論<br/>📡「情報を数値化」
    1950年代 : 最初のプログラミング言語<br/>💻 FORTRAN誕生
    1960年代 : データ構造の体系化<br/>🗂️ 木構造・グラフ理論
    1980年代 : アルゴリズム理論の黄金期<br/>⚡ 計算量理論確立
    2000年代 : ビッグデータ時代<br/>📊 新しいデータ処理技術
    2010年代〜 : AI革命<br/>🤖 ディープラーニング爆発
```

### 📖 重要な転換点

**1. アラン・チューリング（1936年）**
> 「機械は人間のように計算できるか？」という問いから、**チューリングマシン**を考案。これが現代コンピュータの理論的原型に。

**2. クロード・シャノン（1948年）**
> 電話回線のノイズ問題を解決する中で、**情報を数値化する方法**を発見。「1ビット = 2択の情報量」という概念が生まれた。

**3. ドナルド・クヌース（1960年代）**
> アルゴリズムを芸術的に分析する手法を確立。「美しいコード」の基準を作った。
